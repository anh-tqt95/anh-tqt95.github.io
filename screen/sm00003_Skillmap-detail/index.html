    Created User : CSV TAI

ScreenID : SM00003
ScreenName : SKILLMAP DETAIL

    1. Screen Summary

    1.1 Purpose

    Show a list of all employee skills. 
Show employee skill up when viewing timeline.
Show roadmap.

    1.2 Main Function

    Table: sm_skill, sm_skill_category, sm_skill_map, sm_skill_level, sm_roadmap, sm_employee_certificate, sm_skill_document, sm_target, sm_recommended_target

    Request API: 

    - [GET] base_url/api/skillsmap-detail?employeeId={employeeId} (Get all employee skills, levels, certificates, documents and roadmaps)
- [POST] base_url/api/skillsmap-detail/skill/{employeeId} (Create Skill for given employeeId)
- [POST] base_url/api/skillsmap-detail/level/{employeeId} (Create Level for given employeeId)
- [POST] base_url/api/skillsmap-detail/certificate/{employeeId} (Create Certificate for given employeeId)
- [POST] base_url/api/skillsmap-detail/document/{employeeId} (Create Document for given employeeId)
- [POST] base_url/api/skillsmap-detail/roadmap/{employeeId} (Create Roadmap for given employeeId)
- [PUT] base_url/api/skillsmap-detail/skill/{employeeId} (Update Skill for given employeeId)
- [PUT] base_url/api/skillsmap-detail/level/{employeeId} (Update Level for given employeeId)
- [PUT] base_url/api/skillsmap-detail/certificate/{employeeId} (Update Certificate for given employeeId)
- [PUT] base_url/api/skillsmap-detail/document/{employeeId} (Update Document for given employeeId)
- [PUT] base_url/api/skillsmap-detail/roadmap/{employeeId} (Update Roadmap for given employeeId)
- [DELETE] base_url/api/skillsmap-detail/skill/{employeeId} (Delete Skill for given employeeId)
- [DELETE] base_url/api/skillsmap-detail/level/{employeeId} (Delete Level for given employeeId)
- [DELETE] base_url/api/skillsmap-detail/certificate/{employeeId} (Delete Certificate for given employeeId)
- [DELETE] base_url/api/skillsmap-detail/document/{employeeId} (Delete Document for given employeeId)
- [DELETE] base_url/api/skillsmap-detail/roadmap/{employeeId} (Delete Roadmap for given employeeId)
- [POST] base_url/api/skillsmap-detail/timeline?employeeId={employeeId}&skillId={skillId} (Display skill up information of employees)
- [POST] base_url/api/skillsmap-detail/roadmap?roadmapId={roadmapId} (Display roadmap information of roadmap selected)
    BE Function: 
    - Function sendConfirm()
- Function approveConfirm()
- Function sendRejection()

    1.3 Remark

    2. List of contribution (Admin page)

    2.1 layout

    2.2 Description

    Màn hình skill map:     
- Hiển thị thông tin:       
    + Thông tin user    
    + Dự án tham gia (all)  
    + Skill map và mức độ thành thạo    
    + Mục tiêu phát triển   
    + Chứng chỉ 
    + Gợi ý tài liệu, nguồn học tập. (Tự gợi ý theo skill và mục tiêu)  
    + Người chứng nhận (quản lý dự án tham gia) 
    + Chú ý (góp ý từ quản lý)  
    + Roadmap (User tự chọn roadmap cho mình => Từ roadmap có thể hiển thị cho phần gợi ý)  
- Chức năng đăng ký chỉnh sửa ( tùy role thay đổi hiển thị tên Button):     
    + Role là user: Sau khi dự án kết thúc nhân viên cập nhật skill 
        thông báo cấp quản lý xác nhận
    + Role là admin: xác nhận hoặc gửi request lại cho nhân viên    
- Khi nhấn button xem timeline hiển thị Popup timeline của nhân viên đã chọn.
